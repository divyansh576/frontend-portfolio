# ⚡ Pokédex Web Application

A responsive and interactive Pokédex built using HTML, CSS, and JavaScript. This project allows users to search and view Pokémon details in a clean and user-friendly interface.

---

## 🚀 Features
- Search Pokémon by name or ID  
- Dynamic data rendering  
- Responsive design for all screen sizes  
- Clean and interactive UI  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Fetch API 
- DOM Manipulation
